Как сделать .exe файл из программы на Python

Ноябрь 3, 2021
Чтобы создать .exe файл с Вашей python программы, нам необходимо сделать несколько простых действий:

Через поиск ищем командную строку и открываем ее.

Что бы сделать .exe файл из программы на Python необходимо установить библиотеку pyinstaller.

Пишем в командной строке: pip install pyinstaller

После переходим в папку с Вашим файлом .py в командной строке:

Запускаем команду pyinstaller не забывая указать имя вашего скрипта.

pyinstaller --onefile .py

Готово! В папке появится папка src и там будет .exe файл.
